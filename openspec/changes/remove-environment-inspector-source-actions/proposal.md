# Remove Environment Inspector Source Actions

## Summary

Simplify the gathering environment editor's task and hazard right inspector by removing the selected-record source card and the call-to-action button strip below the selected record name.

## Goals

- Remove the Source card from the right inspector on Tasks and Hazards tabs.
- Remove the selected-record action buttons directly below the selected task/hazard name.
- Keep row-level composition actions and overflow menu source navigation available in the task/hazard lists.

## Out of Scope

- Changing Overview inspector content.
- Removing source navigation from composition row overflow menus.
- Changing composition include/remove behavior.
