# Environment Inspector Evidence Table

## Summary

Change the gathering environment editor's task/hazard right-inspector matching evidence from stacked check rows to a compact two-column table.

## Goals

- Show one row each for biome, region, weather, time, and danger in the selected task/hazard inspector.
- Put the evidence dimension name in the left column.
- Put the matching/not-matched value pills for that dimension in the right column.
- Preserve the existing compact chip evidence variant used outside the inspector.

## Out of Scope

- Changing the composition table rows.
- Changing matching semantics or evidence state calculation.
- Changing the runtime/composition layer card.
