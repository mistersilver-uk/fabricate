# Prompt Library

## Next Backlog Task

```text
Use the orchestrator agent to read BACKLOG.md, pick the next ready task,
delegate it to the implementer agent, then once complete pass the result
to the reviewer agent, then if accepted, pass the result to the docs writer agent,
and finally update BACKLOG.md with the outcome.
```

## Next Backlog Task (Parallel)

```text
Create three teams of orchestrator, implementer, reviewer and docs writer agents and find them 3 tasks that are ready,                                                                                                                                       
with no dependencies, and ideally do not require editing the same files. Hand one task to each teams' orchestrator,                                                                                                                                          
and task it with creating a plan to implement the task, which you should then hand to the teams' implementer agent.                                                                                                                                          
Once implementation is complete, pass the result to the teams' reviewer agent. If not accepted, the implementer must                                                                                                                                         
iterate further on the solution before it is passed back to the reviewer. If accepted, pass the result to the docs                                                                                                                                           
writer agent and finally update BACKLOG.md 
```
